##Linux基础命令

ls (列出目录)

-a ：全部的文件，连同隐藏档( 开头为 . 的文件) 一起列出来(常用)
-d ：仅列出目录本身，而不是列出目录内的文件数据(常用)
-l ：长数据串列出，包含文件的属性与权限等等数据；(常用)

cd (切换目录)

cd是用来切换工作目录的命令，切换的是相对路径或者绝对路径
cd .. 表示去到当前目录的上一级目录
cd ~  表示返回根目录，即/root

pwd (显示目前所在的目录)

-P ：显示出确实的路径，而非使用连结 (link) 路径。

mkdir (创建新目录)

-m ：配置文件的权限喔！直接配置，不需要看默认权限 。
-p ：帮助你直接将所需要的目录(包含上一级目录)递归创建起来！

###touch (创建文件)
- t ： 使用指定的日期时间，而非现在的时间

rmdir (删除空的目录)

-p ：连同上一级『空的』目录也一起删除

cp (复制文件或目录)

-f：为强制的意思，若目标文件已经存在且无法开启，则移除后再尝试一次；
-i：若目标档已经存在时，在覆盖时会先询问动作的进行(常用)
-p：连同文件的属性一起复制过去，而非使用默认属性(备份常用)
r：递归持续复制，用于目录的复制；(常用)

rm (移除文件或目录)

-f ：就是强制的意思，忽略不存在的文件，不会出现警告信息；
-i ：互动模式，在删除前会询问使用者是否动作
-r ：递归删除啊！最常用在目录的删除了！这是非常危险的选项！！！

more（一页一页翻动）

空白键 (space)：代表向下翻一页；
Enter         ：代表向下翻『一行』；
q             ：代表立刻离开 more ，不再显示该文件内容

less （一页一页翻动，与more类似。区别是可以往回翻页）

空白键    ：向下翻动一页；
n         ：重复前一个搜寻 (与 / 或 ? 有关！)
N         ：反向的重复前一个搜寻 (与 / 或 ? 有关！)
q         ：离开 less 这个程序；

head（显示文件前几行）

-n : 后面接数字，代表显示几行

tail （显示文件后几行）

-n :后面接数字，代表显示几行。

grep (用于搜索特定字符)

用法：grep [参数] [文件]

find (用于查找特定文件）

用法：find [参数] [文件] 
-name :查找具有指定文件名的文件
-perm :查找具有指定权限的文件

tar （用于压缩和解压缩文件）

用法：tar [参数] [文件]
-c :建立新的压缩文件
-x ：从压缩文件中提取文件
-z :支持gzip解压文件
-v ：显示操作过程
-f ：指定压缩文件

whereis （定位可执行文件、源代码文件、帮助文件在系统中的位置）

用法：whereis [参数][文件或目录]
-b  定位可执行文件
-s  定位源代码文件

which (搜索path变量中某个系统命令的位置)

用法：which [可执行文件名称]

ping （查看网络上另一个主机系统的网络连接是否正常）

用法：ping [参数][主机名或ip地址]
-c 在发送指定数量的包后停止

ifconfig (查看和配置网络设备)

用法：ifconfig [网络设备][参数]
-up 启动指定网络设备/网卡
-down 关闭指定网络设备/网卡
-a  显示全部接口信息

ps （显示当前进程状态）

用法：ps [参数]    
-aux  列出目前所有正在内存中的程序
-ef   显示所有信息，连同命令行

chown (改变文件的拥有着和群组)

用法：chown [选项][所有者]
-R 处理指定目录以及其目录下的所有文件

chmod (用于改变文件或目录的访问权限)

用法：chmod [选项][文件]
-R 处理指定目录以及其子目录下的所有文件
u：目录或者文件的所属的用户
g:目录或者文件的当前的群组
o:除了目录或者文件的当前用户或者群组之外的用户或者群组
a：所有的用户及群组
