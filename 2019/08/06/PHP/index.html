PHP

什么是PHP

​ PHP是嵌入html页面的脚本语言
目前最流行的网站开发语言
几乎所有平台都可运行
很多企业都使用PHP作为开发语言

PHP 格式

php脚本以结束。

PHP标 签：

​ 1） <%
​ %>

2）<script language="php/text">
 </script>

php脚本可以单独存在，也可以与html结合

PHP扩展名

php 文件扩展名规范，一般以.php结尾，但是可以使用其他命名，例如.phtml等。

PHP 后缀

cgi模式
module模式

基于PHP开发的网站如何配置运行环境

1、apache服务器首选
2、php运行环境
3、视情况看是否需要数据库

PHP的输出

echo （空格和括号都可以输出）

输出需要换行，需要添加标签

print （””); print_r; print_f

die(“”) 结束程序

var_dump(显示输出类型)

input

PHP的输入

服务端输入 $_POST $_GET $_REQUEST
客户端输入 //$argc //$argv

PHP变量的数据类型：

布尔类型（bool) 只有两个元素 true,false （不区分大小写）
整形(int) 正数 0 负数
浮点型（float) 小数 复数
字符串（string) ‘asbdh123’ “123” <<<EOF nejwkfn EOF
资源（resource) STDIN STDOUT STDERROR 打开的文件 打开网络连接
空类型（NULL） （不区分大小写））
数组（array) {1，2，3，4，5，6，7，8} {‘name’ => ‘zs’,’age’=18}
对象(class) new person()

PHP中的变量

变量以$符号开始，后面跟着变量的名称
变量名必须以字母或者下划线开始
变量名只能包含字母数字字符以及下划线
变量名不能包含空格

###   PHP中的常量

常量一旦赋值不会改变，尽量大写
常量分为系统常量和自定义常量
常量可以理解为值不会有改变的量

常用的系统常量

_ FILE_ 当前文件的绝对路径

_ DIR_ 当前文件所在的文件夹

_ LINE_ 当前代码所在行数

PHP_VERSION 当前php版本

PHP_OS 当前操作系统

PHP_EOL 当前系统的换行符

DIRECTORY_SEPARATOR 自适应斜杠

单引号不能放变量，放了不会生效
php 中除了单引号，任意字符放入单引号中，将没有任何意义
php中双引号，放入特殊字符，将会有特殊的意义
EOF 中变量有意义，其他的特殊字符，都没有意义

转义： \ 普通字符串变为特殊字符，特殊字符变为普通字符

运算符

就是处理不同类型的数据。

算术运算符：加减乘除，模（除法的余数）、取反、并置
赋值运算符：在PHP 中，基本的赋值运算符是”=“。 它意味看左 操作数被设为右侧表达式的值。也就是说”Sx = 5 “ 的 值 是 5

比较运算符：
因为比较运算就是用来对数字进行比较的，所以说，遇到字符串和数字进行比较时，就是把字符串转成数字，然后再比较
// ==和///===比较特殊，可以用来对两个字符串进行比较，还能比较布尔值。空类型（NULL），数组。不会进行类型转换

错误控制运算符：@表达式：忽略表达式运行错误

字符串操作运算符：

逻辑运算符； x and y 与、

​ x or y 或、

​ x xor y 异或

​ x && y 与

​ x || y 或

​ ! x 非
