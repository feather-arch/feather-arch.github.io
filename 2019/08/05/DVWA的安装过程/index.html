DVWA的安装过程

进入到/var/www目录下，然后wget下载dvwa文件。

cd /var/www
wget https://github.com/ethicalhack3r/DVWA/archive/master.zip

解压master.zip文件到html文件夹下，进入html文件夹下，修改DVWA-master文件名为dvwa.

unzip master.zip -d html/
cd html && mv DVWA-master dvwa

进入dvwa/config文件夹中，把config.inc.php.dist复制一份，命名为config.inc.php

在浏览器中输入自己的ip /dvwa,查看DVWA

进入到/etc文件夹下，把php.ini备份一份为php.ini.bak

cd /etc
cp php.ini php.ini.bak

打开php.ini，把allow_url_include=Off修改为On，保存退出，重启服务

​ vi php.ini（allow_url_include=Off修改为On）
​ systemctl restart httpd

进入/var/www/html/dvwa/config文件夹下，修改config.inc.php

​ cd /var/www/html/dvwa/config

修改config.inc.php,把$_DVWA[ ‘db_password ’ ] = ‘p@ssw0rd’修改为mysql中root用户的密码，填写公钥和私钥。

​ vi config.inc.php
​ public_key: 6LePqhAUAAAAAH6Bn2okO9-8G-zNw46PNhotV4Q6
​ private_key: 6LePqhAUAAAAAMVqhBbFr1NVA9H13FkyfAgBeZYx

保存退出

进入/var/www/html文件中修改dvwa的用户和用户组

cd /var/www/html
chown -R apache:apache dvwa/

重启服务

systemctl restart httpd
