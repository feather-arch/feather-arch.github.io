Linux基础命令

ls (列出目录)

cd (切换目录)

pwd (显示目前所在的目录)

mkdir (创建新目录)

###touch (创建文件)
- t ： 使用指定的日期时间，而非现在的时间

rmdir (删除空的目录)

cp (复制文件或目录)

rm (移除文件或目录)

more（一页一页翻动）

less （一页一页翻动，与more类似。区别是可以往回翻页）

head（显示文件前几行）

tail （显示文件后几行）

grep (用于搜索特定字符)

find (用于查找特定文件）

tar （用于压缩和解压缩文件）

whereis （定位可执行文件、源代码文件、帮助文件在系统中的位置）

which (搜索path变量中某个系统命令的位置)

ping （查看网络上另一个主机系统的网络连接是否正常）

ifconfig (查看和配置网络设备)

ps （显示当前进程状态）

chown (改变文件的拥有着和群组)

chmod (用于改变文件或目录的访问权限)
